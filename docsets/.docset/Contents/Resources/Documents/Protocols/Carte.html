Carte

public protocol Carte

Ce protocol représente une carte de mouvement.

  • init(nom:mouvements:estCouleurJoueur1:)

    Crée une carte.

    Suite à l'initialisation init(nom: nom, mouvements: mouvements, estCouleurJoueur1: estCouleurJoueur1) on a :

    self.nom == nom
self.mouvements == mouvements
self.estCouleurJoueur1 == estCouleurJoueur1

    Les mouvements mouvements sont des tuple de deux entiers (x, y) tels que x appartient à [-2, 2] et y appartient à [-2, 2]. Si ce n'est pas le cas, c'est une erreur fatale.

    Declaration

    Swift

    init(nom: String, mouvements: [(Int, Int)], estCouleurJoueur1: Bool)

    Parameters

    nom

    Le nom de la carte
    mouvements

    Les mouvements que la carte met à disposition
    estCouleurJoueur1

    true si cette carte est de la même couleur que le joueur 1, sinon false
  • nom

    Indique le nom de la carte.

    Declaration

    Swift

    var nom: String { get }

    Return Value

    un String contenant le nom de la carte

  • mouvements

    Indiques quels mouvements cette carte met à disposition.

    Un mouvement est représenté par une position relative au pion sur lequel on va l'appliquer. Un mouvement est donc un tuple de deux entiers (x, y) tels que x appartient à [-2 ; 2] et y appartient à [-2 ; 2].

    Declaration

    Swift

    var mouvements: [(Int, Int)] { get }

    Return Value

    une liste de mouvements

  • estCouleurJoueur1

    Indique si la couleur de la carte correspond à la couleur du joueur 1. Cette méthode sert pour savoir qui commence la partie, en effet, c'est le joueur dont la couleur correspond à la carte flottante qui commence la partie.

    Declaration

    Swift

    var estCouleurJoueur1: Bool { get }
