Joueur

public protocol Joueur

Ce protocol représente un joueur de la partie. Un joueur possède deux cartes et un ensemble de pions.

  • init(estJoueur1:)

    Crée un joueur.

    Suite à l'appel init(estJoueur1: estJoueur1) on a :

    self.estJoueur1 == estJoueur1
// l'accès à self.cartes émet une erreur fatale

    Declaration

    Swift

    init(estJoueur1: Bool)

    Parameters

    estJoueur1

    true si ce joueur est le joueur 1, sinon false
  • estJoueur1()

    Indique si ce joueur est le joueur 1.

    estJoueur1() == true si self représente le joueur 1

    estJoueur1() == false si self représente le joueur 2

    Declaration

    Swift

    func estJoueur1() -> Bool

    Return Value

    true si ce joueur est le joueur 1, sinon false

  • cartes

    Indiques quelles cartes ce joueur possède.

    Dans le cas d'utilisation du get, on s'attend à ce que le retour soit égal à au tuple donné précédemment lors de l'appel du set. Dans le cas où set n'a pas été appelé, alors get émet une erreur fatale.

    Declaration

    Swift

    var cartes: (Carte, Carte) { get set }

    Return Value

    un tuple de deux cartes qui sont les deux cartes que possède ce joueur à ce moment de la partie
